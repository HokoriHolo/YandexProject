Пояснительная записка к проекту Telegram-бота
Цель проекта: Создание многофункционального Telegram-бота с чатом на основе ИИ и автоматической рассылкой погоды. 
Бот позволяет пользователям общаться с ИИ (DeepSeek-R1) и получать ежедневные уведомления о погоде в выбранном городе в заданное время.
Функционал:
Чат с ИИ: Поддержка контекстного диалога через API DeepSeek-R1. История сообщений хранится в памяти (defaultdict).
Рассылка погоды: Интеграция с OpenWeatherMap. Пользователь настраивает город и время уведомлений через FSM (Finite State Machine).
Управление подписками: Включение/отключение рассылки через кнопки.
Технологии:
Backend: Python 3.11+, aiogram 3.x (асинхронная работа).
Внешние API: OpenWeatherMap (погода), DeepSeek-R1 (чат).
Хранение данных: Временное (в памяти), без использования БД.
Архитектура:
Пользователь взаимодействует с ботом через Telegram.
Запросы чата отправляются в DeepSeek-R1 API.
Данные погоды запрашиваются из OpenWeatherMap.
Фоновая задача (check_weather_schedule) проверяет время для рассылки каждые 30 секунд.
Особенности:
Простота настройки (кнопочное меню, FSM для погоды).
Обработка ошибок API (погода, ИИ).
Поддержка Markdown в ответах.
Перспективы:
Добавление БД (PostgreSQL/Redis) для хранения данных.
Расширение функционала (напоминания, курсы валют).
Вывод: Бот сочетает полезные функции (чат + погода) с минимальными требованиями к инфраструктуре. Готов к масштабированию.
